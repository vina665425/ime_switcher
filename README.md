# 繁簡中文輸入切換器

## 功能
- 切換中文輸入法的繁體 / 簡體輸出
- 支援自定義快捷鍵
- 系統匣常駐，方便快速操作

## 使用方法
1. 執行 `ime_switcher.exe`
2. 主視窗顯示目前輸出狀態
3. 點擊 **「切換繁簡中文」** 按鈕即可切換
4. 點擊 **「點我設定快捷鍵」** 設置快捷鍵
5. 縮小視窗後程式會最小化到系統匣
6. 從系統匣右鍵選單可 **「顯示視窗」** 或 **「退出程式」**

## 自定義快捷鍵注意事項
- 設定完成後，程式會自動生成一個 `hotkey.txt` 文件，
  用於保存快捷鍵設定，建議使用資料夾將EXE檔與文件檔一起儲存起來
- 若快捷鍵失效，請重新以管理員身份執行程式

## 系統需求
- Windows 10 / 11
- 無需安裝
